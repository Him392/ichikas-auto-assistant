一歌小助手（简称 iaa）是一款适用于《世界计划：多彩舞台》的自动化日常脚本，基于 kotonebot 自动化框架编写而成，使用 OpenCV 对屏幕画面进行检测来实现自动化。

> [!IMPORTANT]  
> 一歌小助手 iaa **只用于清日常**，不可以也不会支持任何外挂性质的功能（如以冲榜为目的的自动演出）。

## 源码执行
```bash
# 如果你没安装 uv 需要先安装
uv sync
uv pip install -e .
uv run tools/make_resources.py
uv run iaa/main.py
```
